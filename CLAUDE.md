# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository purpose

Personal course notes for a Claude Code course. All notes are written in Ukrainian. No build system, tests, or dependencies exist — this is a pure markdown notes repository.

## Structure

Each module or lecture has its own directory containing a `notes.md` file:

- `lecture-01-what-is-coding-assistant/` — AI coding assistants, tool use, agentic loop
- `module-02-getting-hands-on/` — CLAUDE.md, planning mode, custom commands, MCP
- `module-03-hooks-and-sdk/` — hooks (PreToolUse/PostToolUse), settings.json, SDK usage

New modules follow the same pattern: `<module-name>/notes.md`.

## Language

Notes are written in Ukrainian. When adding or editing notes, continue in Ukrainian.

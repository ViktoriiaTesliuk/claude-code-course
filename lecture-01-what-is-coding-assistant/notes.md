# Лекція 1: What is an AI Coding Assistant?

## Ключові поняття

- **AI Coding Assistant** — система LLM + виконання реальних інструментів
- **Tool Use** — архітектура яка дозволяє моделі читати файли, запускати команди
- **Agentic Loop** — цикл: запит → виклик інструменту → результат → відповідь

## Як вирішує задачі
1. Gather context — читає файли, розуміє кодову базу
2. Formulate a plan — вирішує підхід
3. Take action — редагує, запускає, перевіряє

## Чому Claude кращий за інших
- Оптимізований для tool use
- Розширюється через MCP серверити
- Код залишається локально (безпека)
